# Roster App

This is a simple roster application built using React. It allows users to manage a list of team members, including adding new members, editing existing members, and deleting members from the roster.

## Tech Stack
- MongoDB
- Express
- React.js
- Node.js

## Implemented Features
- Sign up and login
- Verify account via email verification
- Reset password via email
- Create, read, update types of rosters
- Create, read, update, delete rosters (By range)
